Dexie.js

A Minimalistic Wrapper for IndexedDB

Download Fiddle ( only ~16k minified and gzipped )

Easy to learn

Dexie was written to be straightforward and easy to learn. If you've ever had to work with native IndexedDB then you'll certainly appreciate Dexie's concise API.

Dive right in »

Well documented

What good is any development tool without great documentation? Dexie is thoroughly explained, and examples are available to help you on your way.

Check it out »

Extendable

Adding custom methods to your Dexie instance is easy. Simply Dexie.addons.push() and you've got a reusable, chainable method like any other native one.

See it in action »

Basic examples

/* |----------------------------| | Make a database connection | |----------------------------| */ var db = new Dexie('MyDatabase'); // Define a schema db.version(1) .stores({ friends: 'name, age' }); // Open the database db.open() .catch(function(error){ alert('Uh oh : ' + error); });
/* |-----------------------| | Then run some queries | |-----------------------| */ // Find some old friends db.friends .where('age') .above(75) .each(function(friend){ console.log(friend.name); }); // or make a new one db.friends .add({ name: 'Camilla', age: 25 });
